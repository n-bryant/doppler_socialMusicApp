A social music app, connecting you with the latest sounds

What is doppler?

Doppler is a mock-up of a responsive social media app that focuses around sharing songs. Doppler was built using Angular 1.5, compiled with gulp, and uses local storage to cache and display user data. The songs available are selections from a few of my favorite bands. The data for the songs came from Spotify's Web API.

To get started, simply set a username for yourself and submit the login form. To play around with making friends, create a few users, which will be set into local storage and become available during perusal of the app.
